INSTANT START (PYTHON)
════════════════════════════════════════════════════════════════════════════════

1. Make sure Python 3.8+ is installed:
   python --version

2. Run the game:
   python f15_simulator.py

3. Press [S] to start new game

That's it! No compilation needed.

INSTANT START (C++)
════════════════════════════════════════════════════════════════════════════════

Option A: If you have MinGW/g++ installed:

1. Compile:
   g++ -O2 F15_Simulator.cpp -o F15.exe

2. Run:
   F15.exe

3. Press [S] to start new game

Option B: If you don't have a compiler:

1. Download MinGW: https://www.mingw-w64.org/
2. Install (takes 10 minutes)
3. Follow Option A above

Or use online compiler at replit.com/languages/cpp

MODDING QUICK REFERENCE
════════════════════════════════════════════════════════════════════════════════

Python Version (Edit & Run):
  1. Open f15_simulator.py in any text editor
  2. Find the constant or function you want to change
  3. Save the file
  4. Run: python f15_simulator.py
  5. Changes take effect immediately

C++ Version (Edit, Compile, Run):
  1. Open F15_Simulator.cpp in any text editor
  2. Find the constant or function you want to change
  3. Save the file
  4. Compile: g++ -O2 F15_Simulator.cpp -o F15.exe
  5. Run: F15.exe
  6. Changes take effect

Example Modifications:

INCREASE SPEED:
  Python:  Find "MAX_SPEED = 950", change to 1200
  C++:     Find "#define MAX_SPEED 950", change to 1200

ADD ENEMIES:
  Python:  In spawnInitialContacts, change "random.randint(4, 6)" to "random.randint(10, 15)"
  C++:     In spawnInitialContacts, change "4 + rand() % 3" to "10 + rand() % 5"

MORE WEAPONS:
  Python:  Find "self.cannon_ammo = 500", change to 2000
  C++:     Find "int cannon_ammo = 500;", change to 2000

See respective GUIDE files for 30+ more modifications!

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Python: "ModuleNotFoundError: No module named 'msvcrt'"
  -> You're not on Windows. Use C++ version on Windows, or
  -> Modify input handling for non-Windows (see comments in source)

C++: "g++: command not found"
  -> MinGW not installed. Download from mingw-w64.org
  -> Or use online compiler at replit.com

Both: "Game runs too fast/slow"
  -> Change GAME_TICK value (Python) or GAME_TICK_MS (C++)
  -> Default is 50ms (20 FPS). Try 33ms for 30 FPS, 100ms for 10 FPS

Both: "Screen size is wrong"
  -> Default is 80×24 characters
  -> On Windows: Right-click console → Properties → Layout
  -> Set width=80, height=24
