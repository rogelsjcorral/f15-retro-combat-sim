VERSION UPDATE - v1.1
═════════════════════════════════════════════════════════════════════════════════

This version introduces CONTROL AND TIMING CLEANUP.

CHANGES:

INPUT SYSTEM:
  • Added key debounce to prevent rapid toggle repeats
  • Introduced continuous control handling for pitch/roll
  • Separated "held" vs "toggle" inputs

FLIGHT CONTROL:
  • Pitch and roll now respond smoothly to held keys
  • Roll auto-centers when controls are released

STATE MANAGEMENT:
  • Added PAUSED flight phase
  • Pause no longer jumps to mission results

TIMING:
  • time_in_flight now uses real delta time (float)
  • Mission timer reflects actual elapsed seconds

RADAR:
  • Only one target can be locked at a time
  • Radar display now shows LOCK indicator

ARCHITECTURAL NOTE:
  v1.1 does NOT change physics model.
  v1.2 will introduce physics refactor.
