## PHYSICS MODEL (SAME AS PYTHON VERSION)

### Speed Envelope
```cpp
MIN_SPEED = 150      // Knots - stall threshold
MAX_SPEED = 950      // Knots - max level flight
CRUISE_SPEED = 450   // Knots - optimal
STALL_SPEED = 140    // Knots - hard stall
```

Speed is controlled by throttle + drag:
```cpp
float targetSpeed = minThrust + throttle * (maxThrust - minThrust);
if (velocity < targetSpeed) {
    acceleration = THRUST_RESPONSE;  // Accelerate
} else {
    acceleration = -DRAG * (velocity - targetSpeed);  // Decelerate
}
```

### Pitch (Vertical Control)
```cpp
float pitchRad = pitch * PI / 180.0f;
float verticalAccel = sin(pitchRad) * 9.81f;
```

- **Pitch +45°** → Climb (max rate)
- **Pitch 0°** → Level flight
- **Pitch -45°** → Dive (max descent)

### Roll & Turning
```cpp
float rollRad = roll * PI / 180.0f;
float turnRate = sin(rollRad) * TURN_RADIUS_FACTOR * velocity;
heading += turnRate * dt;
```

- **Roll ±45°** → Moderate turn
- **Roll ±90°** → Maximum turn rate
- **Roll 0°** → No turn

### G-Force Damage
```cpp
if (g_force > G_LIMIT) {  // G_LIMIT = 8.0
    damage_level += 0.01f * (g_force - G_LIMIT);
    damaged = true;
}
```

Sustained high-G turns cause structural damage. Reduces aircraft effectiveness.

### Stall Behavior
```cpp
if (currentSpeed < STALL_SPEED) {
    pitch = max(pitch - 5.0f, -45.0f);  // Nose drops
}
```

When stalled, nose drops automatically. Recovery: increase speed, dive to regain airspeed.

### Terrain Elevation
```cpp
float getElevation(float x, float z) {
    float m1 = 200.0f * exp(-((x-5000)^2 + (z-5000)^2) / 5e6);
    float m2 = 150.0f * exp(-((x-15k)^2 + (z-10k)^2) / 8e6);
    return m1 + m2;
}
```

Generates two Gaussian mountain peaks. Safety buffer: aircraft bumps up if within 100m of ground.

## VECTOR3 CLASS (3D MATH)

The Vector3 structure handles all 3D calculations:

```cpp
struct Vector3 {
    float x, y, z;
    
    // Creation
    Vector3(10, 20, 30);
    
    // Operations
    float mag = v.magnitude();           // Vector length
    Vector3 norm = v.normalize();        // Unit vector
    float dot = v.dot(v2);              // Dot product
    Vector3 sum = v + v2;               // Addition
    Vector3 diff = v - v2;              // Subtraction
    Vector3 scaled = v * 2.5f;          // Scalar mult
};
```

All position, velocity, and acceleration calculations use Vector3.

## GAME STATE STRUCTURE

### Aircraft State
```cpp
struct Aircraft {
    Vector3 position;      // x, y (altitude), z in meters
    Vector3 velocity;      // vx, vy, vz in m/s
    float pitch;           // -90 to +90 degrees
    float roll;            // -180 to +180 degrees
    float heading;         // 0-360 degrees (compass)
    float throttle;        // 0.0 to 1.0
    
    float g_force;         // Current G loading
    bool damaged;          // Took battle damage?
    float damage_level;    // 0.0 (healthy) to 1.0 (destroyed)
    
    // Weapons
    int cannon_ammo;       // 500 rounds
    int missiles_heat;     // AIM-9 Sidewinder (4)
    int missiles_radar;    // AIM-7 Sparrow (4)
    int missiles_agm;      // AGM-65 Maverick (8)
    int bombs;             // Mk82 (12)
    
    // Helper functions
    float currentSpeed()   // Returns speed in knots
    float currentAltitude() // Returns altitude in feet
    bool isStalled()       // Check stall condition
};
```

### Contact (Radar) Structure
```cpp
struct Contact {

    int threat_level;      // 0-9 scale

};
```

## WINDOWS CONSOLE API (Unique to C++)

The C++ version uses Windows Console directly for MAXIMUM SPEED and authenticity:

```cpp
// Get console handle
HANDLE hStdOut = GetStdHandle(STD_OUTPUT_HANDLE);

// Clear screen
FillConsoleOutputCharacterA(hStdOut, ' ', WIDTH*HEIGHT, ...);

// Set cursor position
COORD coord = {x, y};
SetConsoleCursorPosition(hStdOut, coord);

// Hide cursor
CONSOLE_CURSOR_INFO info;
info.bVisible = FALSE;
SetConsoleCursorInfo(hStdOut, &info);
```

This is MUCH faster than clearing terminal with system("cls").

### Keyboard Input
```cpp
// Non-blocking key check
if (GetAsyncKeyState(VK_KEY) & 0x8000) {
    // Key is pressed
}
```

No waiting for user input—game remains smooth at 20 FPS.

```cpp
bool attemptLanding(Aircraft& a, const Terrain& t) {
    float terrainElev = t.getElevation(a.position.x, a.position.z);
    
    if (a.position.y - terrainElev < 50) {  // Within 50m of ground
        if (a.currentSpeed() < 200 && a.pitch < 0) {  // Correct approach
            a.on_ground = true;
            a.velocity = Vector3(0, 0, 0);
            a.landing_distance = 0.0f;
            return true;  // Successful landing
        } else {
            a.damage_level = 1.0f;  // CRASH!
            return false;
        }
    }
    return false;
}
```

#### 2. Add Wind Effects

```cpp
#define WIND_SPEED 15.0f  // knots
#define WIND_DIRECTION 270.0f  // degrees

void applyWind(Aircraft& a) {
    float windRad = WIND_DIRECTION * 3.14159f / 180.0f;
    a.velocity.x += cosf(windRad) * WIND_SPEED * 0.5f;
    a.velocity.z += sinf(windRad) * WIND_SPEED * 0.5f;
}
```

Call in `updateAircraft()` after position update.

#### 3. Missile Guidance System

```cpp
struct Missile {
    Vector3 position;
    Vector3 velocity;
    Contact* target;
    float time_remaining;
};

void updateMissiles(Missile* missiles, int count, float dt) {
    for (int i = 0; i < count; i++) {
        if (missiles[i].target != nullptr) {
            // Home toward target
            Vector3 toTarget = missiles[i].target->position - missiles[i].position;
            Vector3 direction = toTarget.normalize();
            missiles[i].velocity = direction * 500.0f;  // 500 m/s
            missiles[i].position = missiles[i].position + missiles[i].velocity * dt;
            
            // Check impact
            if (toTarget.magnitude() < 100) {
                missiles[i].target->threat_level -= 5;
                missiles[i].time_remaining = 0;  // Remove missile
            }
        }
    }
}
```

## TROUBLESHOOTING

### "cl is not recognized"
You don't have MSVC installed. Use MinGW instead:
```bash
g++ -O2 F15_Simulator.cpp -o F15.exe
```

### "g++ is not recognized"
Install MinGW from: https://www.mingw-w64.org/

### Game runs in fullscreen/wrong size
The console is set to 80×24. Windows may auto-scale. Try:
- Right-click window → Properties → Layout
- Set width: 80, height: 24

### Screen flickers constantly
This is intentional (DOS-era 20 FPS). To change:
Find `GAME_TICK_MS 50` and change to:
- 33 = 30 FPS (smoother)
- 100 = 10 FPS (choppier/more "DOS")

### Input lag or no response
Game runs at 20 FPS with 50ms frames. If input feels sluggish:
1. Close other programs
2. Check CPU usage
3. Recompile with -O3 flag
